Operators in JS

Arithmetic operators

JavaScript provides several arithmetic operators to perform arithmetic operations on numbers. These operators are + (addition), - (subtraction), * (multiplication), / (division), % (modulus), ++ (increment), and -- (decrement) 12345. Here is a brief description of each operator: +: Adds two numbers or concatenates two strings.
-: Subtracts one number from another.
*: Multiplies two numbers.
/: Divides one number by another.
%: Returns the remainder of a division operation.
++: Increments a number by 1.
--: Decrements a number by 1.

Assignment operators

The assignment operator in JavaScript is represented by the symbol =. It is used to assign a value to a variable or property. The assignment expression itself has a value, which is the assigned value. This allows multiple assignments to be chained in order to assign a single value to multiple variables . it also has the concept of compound assignment which is combination of arithmetic and assignment operators.

Comparision operators

JavaScript provides several comparison operators to compare values. These operators are used to test for true or false. Comparison operators are used in logical statements to determine equality or difference between variables or values. The comparison operators return a boolean value, either true or false. The == operator compares only the values, while the === operator compares both the values and the types of the operands. The != operator returns true if the operands are not equal, while the !== operator returns true if the operands are not equal in value or not of the same type. The > and < operators compare the values of the operands, while the >= and <= operators compare the values of the operands and return true if the operands are equal to or greater/less than the other operand.

Logical operators

Logical operators in JavaScript are used to perform logical operations and return a boolean value, either true or false. There are four types of logical operators in JavaScript: || (OR), && (AND), ! (NOT). They can be applied to values of any type, not only boolean3. Some examples of logical operators in JavaScript are: true || false returns true
false && true returns false
!true returns false
null ?? "default" returns "default"

Unary operators

The unary plus (+) operator precedes its operand and evaluates to its operand but attempts to convert it into a number,
if it isn't already.

Ternary operators

The conditional (ternary) operator is the only JavaScript operator that takes three operands: a condition followed by a question mark (?), then an expression to execute if the condition is truthy followed by a colon (:), and finally the expression to execute if the condition is falsy. This operator is frequently used as an alternative to an if-else
